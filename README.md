# docmanage
> [!WARNING]
>
> This tool is not finished, and is still in development, and is not currently in a usable state!

Docmanage is an industry-grade markdown document management tool, featuring **safe redaction** and **512-bit encryption**.

## Requirements
* **Operating System:** Linux (any distribution; debain preffered)
* **RAM:** 256MB for the app
